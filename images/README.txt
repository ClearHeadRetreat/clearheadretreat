IMAGE PLACEHOLDERS — Clear Head Retreat
=======================================
Every grey hatched box on the site is a placeholder. To swap one for a real
photo, find the matching <div class="ph ...">LABEL</div> in the HTML and replace
the whole div with:  <img src="images/your-photo.jpg" alt="Describe the photo">

Aspect ratios used:
  ph-hero  -> 4:5  portrait  (hero + host portrait)
  ph-tall  -> 3:4  portrait  (interiors, detail shots)
  ph-wide  -> 16:9 landscape (activity cards)

Shot list:
  1. index hero       : Tongariro River or bush track at dawn (4:5)
  2. index activities : walk, fishing, golf, biking, horse riding, alpine (16:9)
  3. index why-three  : shared table or chairs by a fire (3:4)
  4. retreat stay     : warm interior of the home (3:4)
  5. retreat included : detail — boots by the door, rod, mug (3:4)
  6. host hero        : relaxed outdoor portrait, not a corporate headshot (4:5)
  7. og-cover.jpg     : 1200x630 social share image

For the handmade feel: real, slightly imperfect photos beat polished stock.
Keep colours muted and natural; overcast light suits this palette.
